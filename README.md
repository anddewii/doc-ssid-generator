# DOC SSID Generator - Streamlit

Aplikasi gratis untuk generate PDF Perubahan SSID AP1 dari Google Sheet + Google Drive.

## Deploy ke Streamlit Community Cloud

1. Upload semua file ini ke GitHub repository.
2. Buka https://share.streamlit.io
3. Login dengan GitHub.
4. Pilih repository ini.
5. Main file path: `app.py`
6. Klik Deploy.

## Syarat

- File di folder site harus mengandung:
  - `_before`
  - `_after`
  - `grafik`

## File penting

- `app.py`
- `requirements.txt`
- `TAMPLATE PERUBAHAN SSID AP1.pdf`
